1 概概述述
基因组survey以测序技术为基础，基于小片段文库的低深度测序，通过K-mer分析，快速获得基因组大小、杂合度、重复序列比例等基本信息，为制定该物种的全基
因组de novo测序策略提供有效依据。
2 实实验验方方法法
检测合格的DNA样本通过Covaris超声波破碎仪随机打断成长度为300-500bp的片段，经末端修复、加A尾、加测序接头、纯化、PCR扩增等步骤完成整个文库制备。
构建好的文库进行二代PE测序。
3 Survey 分分析析流流程程
基因组survey分析流程主要包括：数据质控、NT比对、K-mer分析等步骤，得到clean数据、评估数据的污染情况、预估基因组大小、杂合度、重复率等信息。

4 测测序序数数据据产产出出及及质质控控
检测合格的DNA样本通过Covaris超声波破碎仪随机打断成长度为300-500bp的片段，经末端修复、加A尾、加测序接头、纯化、PCR扩增等步骤完成整个文库制备。
构建好的文库通过Illumina NovaSeq或BGI MGISEQ进行PE150测序。
测序数据的产生经过了DNA提取、建库、测序多个步骤的，然而，这些步骤中产生的无效数据会对生物信息数据高级分析带来严重的干扰，比如建库阶段会出现建
库长度的偏差，测序阶段会出现测序错误的情况。我们必须通过一些手段将这些无效数据过滤排除掉，以保证生物信息分析的正常进行。
4.1 数数据据说说明明
4.1.1 原原始始测测序序数数据据
高通量测序 ( 如BGI的MGIseq2000等一系列的测序平台 ) 测序得到的原始图像数据文件经碱基识别 ( Base Calling ) 分析转化为原始测序序列 ( Sequenced Reads ) ，我
们称之为 Raw Data 或 Raw Reads，结果以 FASTQ ( 简称为 fq ) 文件格式存储，其中包含测序序列 ( reads ) 的序列信息以及其对应的测序质量信息。测序样本中真实数据
随机截取结果如下：

每个序列共有 4 行信息：
第 1 行是序列名称，由测序仪产生，包含index序列及read1或read2标志；
第 2 行是序列，由大写“ACGTN”组成；
第 3 行是序列ID，也有省略了ID名称后直接用“+”表示；
第 4 行是序列的测序质量，每个字母对应第 2 行每个碱基；
测序碱基质量值的表示方法：
碱基的质量都是以ASCII值表示的，根据测序时采用的质量方案的不同，计算十进制的质量值的方法也有所区别，常见的计算方法如下所示：
（1）质量值 = 字符的ASCII值 - 64
（2）质量值 = 字符的ASCII值 - 33
测序碱基质量值的范围是[0,40]，即ASCII值表示为[B,h] 或 [#,I]。
测序错误率与测序质量值简明对应关系。具体地，如果测序错误率用 E 表示，碱基质量值用 Q 表示，则有如下关系 ： Q = -10 log10(E)。
4.1.2 Clean Data
原始测序数据中会包含接头信息，低质量碱基，未测出的碱基 ( 以N表示 ) ，这些信息会对后续的信息分析造成很大的干扰，通过精细的过滤方法将这些干扰信息去
除掉，最终得到的数据即为有效数据，我们称之为clean data 或 clean reads，该文件的数据格式与Raw data完全一样。
4.2 数数据据过过滤滤方方法法
测序得到的原始序列，可能带有接头和低质量的reads。为了保证信息分析质量，首先需要对Raw reads进行过滤，得到Clean reads，后续分析基于Clean reads进行。
我们采用SOAPnuke软件对Raw reads进行过滤，得到Clean reads，主要参数：–lowQual=20，–nRate=0.005，–qualRate=0.5，其他参数默认。数据处理的步骤如下：
1）过滤掉含有接头序列的reads；
2）去除由于PCR扩增等原因引起的duplicated reads；
3）当单端测序read中的一端含有的N的含量超过该条read长度比例的 10% 时，需要去除此对paired reads；
4）当单端测序read中的一端含有的低质量 ( <=5 ) 碱基数超过该条read长度比例的 50% 时，需要去除此对paired reads。

4.3.2 测测序序质质量量评评估估
使用FastQC软件，从以下四方面对测序质量进行评估：
1）测序数据质量检查：测序的质量应该主要分布在Q20以上，这样才能保证后续高级分析的正常进行。
2）测序错误率分布检查: 根据测序技术的特点，测序片段末端的碱基质量一般会比前端的低。测序错误率与碱基质量有关，根据测序技术的特点，测序片段末端
的错误率会偏高，并且大片段文库测序的错误率比小片段文库测序的错误率高。
3）碱基频率分布检查：用于检测有无AT、GC分离现象。统计测序碱基A、T、C、G、N的含量分布，根据碱基互补原则，A和T的比例应该接近，C和G的比例也
应该是接近的。而N含量也反映了测序质量的好坏。
4）GC含量分布检查：通常GC含量应该呈现单峰，如果存在多峰，可能由外源基因组污染引起。

5 K-mer分分析析
5.1 K-mer 分分析析原原理理
基因组在组装前，可以通过测序得到的序列估计其特征。我们采用基于K-mer 的分析方法来估计基因组大小和杂合率以及重复序列信息。
K-mer：即从一段连续序列中迭代地选取长度为K个碱基的序列，若每条序列的长度为L，那么可以得到L-K+1个K-mer。我们通常取K=17来进行分析。( 针对长度为
K的K-mer，对于 A,T,C,G四种碱基类型，一共能产生的K-mer种类数为4k个；因此，选择K=17，可以产生的K-mer 种类数为 417，由此可保证产生足够多的K-mer种类数
去覆盖整个基因组。)

大小。
